Ubuntu16.04服务器搭建ss服务

Ubuntu16.04服务器搭建ss服务

  1. 安装unzip wget

    sudo apt-get install unzip wget

  2. 安装shadowsocks

    wget https://github.com/ToyoDAdoubi/doubi/archive/master.zip
unzip master.zip
cd ./master
sudo chmod 777 ssr.sh
sudo ./ssr.sh

  3. 安装过程一路都有提示，十分简单
